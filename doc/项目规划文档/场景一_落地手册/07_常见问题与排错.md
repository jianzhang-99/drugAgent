# 07 常见问题与排错

## 1. 提交任务后进度一直 0%

排查顺序：
1. 是否真正启动了异步执行线程
2. 任务状态是否写回共享仓库
3. 前端轮询 `taskId` 是否正确

## 2. 向量检索结果为空

排查顺序：
1. 基准文档是否完成入库
2. 切片内容是否为空或过短
3. Embedding 调用是否成功

## 3. 大模型输出不是 JSON

排查建议：
1. Prompt 明确要求“仅输出 JSON，不要解释文字”
2. 服务端加一层 JSON 解析容错与重试
3. 失败时保留原始响应用于定位

## 4. 前端显示“网络异常”

排查顺序：
1. 后端是否运行在 `8124`
2. 前端代理是否将 `/api` 转发到后端
3. 浏览器 Network 面板检查真实请求路径

## 5. 高亮位置对不齐

排查建议：
1. MVP 先做段落级，不做字符级
2. 后端返回段落 ID + 前后文片段
3. 前端按段落匹配渲染，避免精确索引偏移
