# 场景一落地手册总导航（从 0 到可运行）

> 适用对象：第一次做 Agent + RAG + 长文档查重场景的开发者  
> 项目场景：医疗招投标标书雷同与语义查重  
> 当前仓库：`drug-agent`（Spring Boot）+ `drug-agent-web`（Vue3）

## 1. 你将完成什么

完成一个可演示的最小闭环（MVP）：
- 上传待查标书与基准标书
- 后端异步处理（解析 -> 切片 -> 检索 -> 重排 -> LLM 裁决）
- 返回结构化报告
- 前端展示任务进度与双栏高亮结果
